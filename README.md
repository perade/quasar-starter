## Setup

#### Prerequisite

- Node: >= 14.17.0
- Npm: >= 6.14.13
- Yarn: >= 1.22.10

#### Installation

```bash
# if you need, run for each package.json
yarn
```

### Usage

#### Start

```bash
# SPA
yarn dev

# Android
yarn dev:aos

# iOS
yarn dev:ios

# Electron
yarn dev:electron
```

#### Build
```bash
# SPA
yarn build

# Android
yarn build:aos

# iOS
yarn build:ios

# Electron
yarn build:electron
```
